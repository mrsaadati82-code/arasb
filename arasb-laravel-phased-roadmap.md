# نقشه راه فازبندی‌شده اجرای ARASB بر بستر Laravel + MySQL + Shared Hosting

## 1) هدف سند
این سند، مسیر اجرایی پروژه ARASB را با توجه به شرایط واقعی هاست Linux5 ایران‌سرور مشخص می‌کند تا پروژه به‌صورت فازبندی‌شده، قابل کنترل، قابل نصب و قابل تحویل پیش برود.

این نقشه راه بر این مبناست:
- هاست: Linux5 ایران‌سرور
- کنترل‌پنل: DirectAdmin
- دیتابیس: MySQL
- سمت سرور: PHP 8.1+
- نیاز به فایل zip برای نصب/آپدیت
- پنل مدیریت فارسی
- سایت کاربر 3 زبانه
- حفظ کامل ظاهر فعلی سایت

---

## 2) اصل راهبردی پروژه
پروژه باید بر اساس این سه اصل جلو برود:
